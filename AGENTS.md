# AGENTS.MD

### Commit Convention

- `type(scope): commit message (ticket_number)` 커밋 메시지 컨벤션을 준수하세요.
- 가능한 type: feat, fix, refactor, comment, docs, test, chore
- 하나의 커밋은 하나의 목적만 담습니다.
- 커밋 메시지는 변경 내용을 명확하게 드러내도록 작성합니다.
- Jira 티켓 번호는 반드시 포함합니다.
- 너무 긴 메시지보다, 핵심 변경 사항이 드러나는 짧고 명확한 문장을 사용합니다.

- 필수는 아니지만, 커밋 메시지는 한국어 작성을 권장합니다.
